Folders, or individual clips. BRAW, R3D, ARRIRAW/MXF, MP4, MOV all accepted.

Checksums (SHA-256) are computed fully, in-browser, in chunks — safe for large RAW files. Resolution, duration and preview frames auto-fill for standard MP4/MOV where the browser can decode it natively. For codecs the browser can't decode on its own (e.g. ProRes, H.265 on some systems), this page automatically falls back to an embedded FFmpeg engine (~31MB, loads once, cached after) — this only works when the page is opened via its hosted link (https://…), not a downloaded local copy, and only for files under 1.5GB. BRAW / R3D / ARRIRAW can't be decoded by any browser-based method, FFmpeg included — camera fields and previews for those need manual entry, and values highlighted amber are guessed from the filename, so double-check them.
